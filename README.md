# Jorge Armando Ramírez Díaz — Academic Website

Personal academic website for Jorge Armando Ramírez Díaz, built with Astro and adapted from the Astro Scholar theme.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production site is deployed to GitHub Pages via `.github/workflows/deploy.yml`.

## Content

- Global metadata and links: `src/consts.ts`
- Publications: `src/data/publications.bib`
- Projects: `src/data/projects.json`
- News posts: `src/content/blog/`
- CV file: `public/cv-jorge-armando-ramirez-diaz.pdf`
